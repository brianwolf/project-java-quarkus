# java-quarkus

> Ejemplo de proyecto de java 17 con quarkus

![alt](docs/img/quarkus_logo.png)


## :gear: Requisitos

* openjdk 17
* Apache Maven 3.8.4 

## :tada: Uso

```bash

# correr usando dev
make r

# compilar a binario
make p
```
